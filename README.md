# Two-Player Trivia Battle Game
A browser-based **two-player trivia game** built using **JavaScript**, where players compete in multiple rounds by answering trivia questions of varying difficulty. The game fetches real-time questions using **The Trivia API**.


## Features
- Two-player turn-based gameplay  
- Multiple rounds with unique categories  
- Difficulty-based scoring system  
- Real-time trivia questions via API  
- Final winner or draw declaration  


## Tech Stack
- HTML  
- CSS (basic styling)  
- JavaScript (Vanilla)  
- The Trivia API  

**API Documentation:**  
https://the-trivia-api.com/docs/v2/


## Game Flow (Brief)
1. Players enter unique names  
2. Select a category for each round  
3. Answer 6 questions per round (Easy → Medium → Hard)  
4. Scores update based on difficulty  
5. Continue to the next round or end the game  
6. Final scores decide the winner  


## Scoring Rules
- Easy → 10 points  
- Medium → 15 points  
- Hard → 20 points  
- Wrong answer → 0 points
